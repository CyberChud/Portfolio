Project 1: This portfolio site

Building My Cybersecurity Portfolio Website

In this post, I be documenting the process of building my cybersecurity portfolio website and the skills gained to create it, this will be an ongoing project as I learn different concepts and features. I will continue to update this post with updates that I make to this site.

Project Overview

The goal of this project was to create a professional and visually appealing portfolio website that highlights my cybersecurity skills and experiences. The site includes several key sections: an overview of my skills, detailed descriptions of my projects, my professional certifications, and a contact form for potential employers to reach out to me.

Skills Demonstrated

Building this website required a combination of technical and creative skills, including:

  • HTML & CSS: Structuring the content and styling the pages to ensure a clean, professional look. This included using Tailwind CSS for responsive design and easy styling.
  • JavaScript: Adding interactive elements to enhance the user experience. This included using animations and form handling techniques.
  • Responsive Design: Ensuring the website looks good on all devices, from desktop monitors to mobile phones. This involved using media queries and responsive units in CSS.
  • Web Hosting: Deploying the website using Netlify, a popular platform for hosting static websites. This also included setting up a custom domain.

Building the Site

The process of building the site involved several steps:

1. Planning and Design

I started by sketching out the layout and structure of the website. This included deciding on the key sections and how to organize the content. I used design tools to create a mockup of the site.

2. Coding the HTML and CSS

Once the design was finalized, I began coding the HTML and CSS. I used semantic HTML to ensure the content is well-structured and accessible. Tailwind CSS was used to style the site and ensure it is responsive.

3. Adding Interactivity with JavaScript

Next, I added interactivity to the site using JavaScript. This included form validation, smooth scrolling, and other interactive elements to enhance the user experience.

4. Deploying the Site

Finally, I deployed the site using Netlify. This involved setting up a GitHub repository, connecting it to Netlify, and configuring the custom domain.

Conclusion

Creating this portfolio website was a valuable learning experience that allowed me to demonstrate my skills in web development and cybersecurity. This will be something I will continously update with new skills that I learn. Thank you for taking the time to visit my portfolio!
